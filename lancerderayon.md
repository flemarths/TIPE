#### Q2: *Écrire une fonction d'entête*

```
def ps(v1,v2):
    return np.inner(v1,v2)
```

#### Q3: *Écrire une fonction d'entête*

```
def norme(v):
	return np.sqrt(ps(v,v))
```

#### Q4: *Écrire une fonction d'entête*

```
def unitaire(v):
	return (1/norme(v))*v
```
